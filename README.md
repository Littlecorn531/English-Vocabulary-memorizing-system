# 📓 高中英语自适应背词系统 (v6.0) 说明文档

本项目是一款基于 Web 架构的轻量级英语词汇辅助学习工具。系统主要面向高中阶段英语词汇的日常检测与复习，侧重于词性转换、词根词缀派生关联的训练。

---

## ⚙️ 系统设计与核心机制

### 1. 记忆追踪机制 (基于 SM-2 算法变体)
* **动态参数调度**：系统为每个单词维护记忆间隔（Interval）和难度因子（Ease Factor）。
* **自适应调度**：每次测试提交后，后端会根据答题正确与否动态更新复习周期，确保已届复习周期的单词在抽词时获得更高的权重，辅助对抗遗忘。

### 2. 词汇测试与交互设计
* **题型模式**：系统提供“看词辨义”、“完形拼写”、“词形变化”及“混合挑战”四种模式。
* **独立拼写区域**：在拼写类题型（完形拼写、词形变化）中，句子中的被测词呈现为静态的虚线槽（附带字母数提示及首字母提示）。用户在句子下方的独立输入框中键入完整的单词拼写进行提交。
* **自定义自动提示**：系统支持用户自定义无操作时的自动释义提示延迟时间（单位为秒），用户亦可选择关闭此自动提示。
* **选项原词解析**：在多选模式（看词辨义）中若回答错误，系统会展示四个备选选项分别对应的英文原词、中文释义和标准词性，用以辅助错题分析。

### 3. 语法形态自适应匹配
* **句境变形兼容**：在拼写类测试中，系统能够通过正则匹配捕获句子中实际所需的变形形式（如复数 `-s/-es`、过去式 `-ed`、现在分词 `-ing` 等），并以此变形作为正确的判定标准，避免了因强制比对词干原形而导致的语法冲突。

### 4. 数据 management 与导入导出
* **日志累加统计**：系统记录单次测试的耗时、答题量及错误分布，并在日志页面累加统计历史练习总次数、总刷题量、实际错误率及累计用时（换算为时、分、秒格式）。
* **格式化导入**：支持读取并解析特定格式的 `.json` 或 `.txt` 本地文件，导入时可自动建立派生词的双向关联。
* **数据重置与导出**：提供一键清空历史缓存及配置的功能。同时支持将高频错词列表导出为本地 Markdown 格式的复盘清单（`Weak_Words_Report.md`）。

### 5. 界面主题与进程管理
* **主题切换**：提供暗色模式与柔雾粉日间模式两套视觉主题，操作区域采用分段选择器设计。
* **手动终止进程**：移除了后台心跳检测自动退出的机制。在侧边栏左下角设计了“终止系统进程”按钮，用户点击并确认后，网页端将向后端发送终止信号，使 Python 后端进程及命令行窗口安全退出。

## 📂 本地数据文件说明

程序运行后，会在同级目录下自动创建并管理以下用于持久化存储的数据文件：
* `vocab_stats.json`: 存储各单词的历史测试频次、错误分布、记忆周期及下次建议复习日期。
* `study_logs.json`: 记录历次练习的测试时间、模式、题量、错误数及秒级耗时。
* `vocab_config.json`: 保存用户上一次选择的词表小节范围，以便下次启动时自动恢复配置。
* `custom_lists.json`: 存放通过外部导入渠道读入的自定义词表数据。
