# 📓 高中英语自适应背词系统 (v6.0)

本系统是一款专为高中英语词汇特训设计的 Web 轻量级交互学习工具。系统特别针对高中英语中的重难点——“词性转换”与“派生词关联”设计，内置艾宾浩斯记忆模型（SM-2 算法），通过数据级前后端联协，实现高强度自适应抽测。

---

## 🛠️ 系统架构与核心机制

### 1. 记忆追踪系统 (SM-2 间隔重复算法)
*   **自适应调度**：系统为每个单词维护记忆间隔（Interval）和难度因子（Ease Factor）。每次测试提交后，后端会实时更新参数，确保到期复习的单词优先被抽取，实现抗遗忘特训。

### 2. 沉浸式“线上直接填空”系统 (Inline Cloze Engine)
*   **行内嵌入输入框**：拼写填空与语法填空突破了传统的独立输入框限制，直接在例句空缺处嵌入高亮、透明底的行内输入框（Inline Input）。
*   **分段字数横线**：空缺处使用 CSS 渐变渲染出等宽、独立的点面虚线。每个下划线代表一个字母。
*   **首字母免输**：在完形填空模式中，单词首字母会作为静态文本直接展示在输入框左侧。使用者无需重复输入首字母，输入框自动对焦在后方空缺处。

### 3. 多选模式错题原词解析
*   **选项溯源**：在“看词辨义”练习中，若使用者答错，系统会自动加载这 4 个备选选项分别对应的英文原词、中文释义和标准词性，帮助使用者理清混淆概念。

### 4. 统计日志数据中心
*   **高维累加统计**：系统在记录单次练习数据的同时，会保存原始秒级耗时。数据中心支持分页浏览完整历史日志，并在顶部实时统计累计练习次数、总刷题量、错误率以及总用时（换算为时、分、秒）。
*   **自动文件格式识别**：导入器支持直接读取并无损合并 `.json` 文件或五段式 `.txt` 本地路径，自动处理派生词双向绑定。
*   **物理重置与错词本导出**：支持一键清空所有历史缓存，以及将弱项错词列表一键导出为本地 Markdown 报告（`Weak_Words_Report.md`）。

### 5. 亚克力 macOS 级视觉主题
*   **色彩主题自适应**：系统内置“夜间模式”与“粉色日间”双套半透明高透光主题。主题选择、测试题型等操作区全面升级为 macOStrans trans Translucent 胶囊分段选择器（Segmented Control）。
*   **看守狗自动进程管理**：系统后台设有守护线程，当检测到浏览器网页端关闭超过 8 秒未发送心跳包时，控制台中的 Python 进程将自动释放端口并安全退出。

## 📂 本地持久化文件清单

运行程序后，系统会在同级目录下自动创建并管理以下数据：
*   `vocab_stats.json`: 存储单词的历史测试频次、错误分布，以及记忆周期、下次复习日期。
*   `study_logs.json`: 记录每次练习的发生时间、模式、做题量、错题数、格式化耗时与原始秒数。
*   `vocab_config.json`: 保存上一次使用的词库小节，支持一键恢复历史选择。
*   `custom_lists.json`: 存放通过外部导入的自定义词表。
