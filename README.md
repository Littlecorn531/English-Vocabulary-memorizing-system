# ⚡ 高中英语智能自适应背词系统 (v6.0 - 状态取反版)

本系统是一款专为高中英语“词性转换”与“派生词关联”设计的终端控制台（Terminal）自适应词汇强化工具。系统在 v6.0 版本中深度重构了词库选择逻辑，并引入了遗忘曲线追踪算法与智能文本模糊匹配技术，旨在为使用者提供高效率、系统化的词汇复盘方案。

---

## 🎯 核心机制设计

### 1. 遗忘曲线追踪 (SM-2 间隔重复算法)
* **动态周期复习**：系统内置简化的 SM-2 记忆算法。根据使用者的答题正确率，动态计算每个单词的记忆间隔（Interval）与难度因子（Ease Factor）。
* **加权高频复现**：系统会自动识别并提示“今日到期需复习”的词汇量。答错的词汇其复习间隔将被重置，抽题权重自动成倍提升，直至使用者彻底掌握。

### 2. 状态取反（Toggle）词库选择面板
* **可视化状态标识**：词库选择列表采用 `[X]`（已勾选）与 `[ ]`（未勾选）直观展示每个子词表的激活状态。
* **序号状态取反**：输入对应序号（如 `1 3`）不再是简单地覆写选择，而是对这几个词表的状态进行取反（切换勾选状态）。
* **区间取反支持 `[l, r]`**：支持输入类似 `[1, 3]` 或 `[2, 5]`（忽略空格）的区间指令，对该区间内的所有词表状态进行一键反转，极大简化了多词表的筛选流程。
* **回车一键同步**：完成勾选后直接敲击回车（Enter），系统会自动保存当前勾选状态作为下次默认配置并退出面板。

### 3. 双格式自适应智能导入器
* **格式自动感应**：系统能自动识别输入文件是标准 JSON 还是 TXT 文本。
* **无损数据合并**：导入标准 `.json` 词库时，系统会将其无缝合并至用户自定义词库（`custom_lists.json`），自动过滤重复词条，并重建派生词的双向网状关联链，规避了传统文本切分导致的数据错乱。

### 4. 语境精准解析与拼写容错
* **正则全字匹配**：完形拼写与语法填空题目中的句子替换采用正则表达式边界匹配（`\b`），可防止因派生词包含相同子串而导致的局部嵌套替换错误，并支持首字母及全大写的智能继承。
* **模糊拼写容错**：在拼写模式下，当输入相似度达到 85% 以上（如错漏个别不改变词根的字母）时，系统会触发黄色手误警告并展示标准拼写，降低挫败感。
* **无干扰选项辨析**：去除了看词辨义模式下干扰选项里的括号词性标签（如 `[n.]`），防止使用者利用语法规律排除干扰项，保证词义测试的纯粹性。

## 🗳️ 词库选择操控指令说明

进入【选项 7】后，通过以下指令对词库进行状态管理：

| 指令/输入 | 作用效果 | 示例 |
| :--- | :--- | :--- |
| **`A` / `a`** | 一键勾选列表中所有的词表 | `A` |
| **`N` / `n`** | 一键取消勾选列表中所有的词表 | `N` |
| **`回车` (Enter)** | 保存当前勾选状态，加载对应词池并返回主控制台 | *(直接按回车)* |
| **`数字序号`** | 对指定序号的词表进行勾选状态取反（支持多选） | `1 3 5` (反转 1, 3, 5 的状态) |
| **`[l, r]`** | 对指定区间内的所有词表状态进行一键反转 | `[1, 3]` 或 `[2, 5]` |
| **`混合指令`** | 支持单号与区间指令混合输入 | `1 [3, 5] 7` |
| **`i` / `I`** | 调用智能导入器加载本地外部词库 | `i` |

## 📂 本地持久化文件清单

系统在同级目录下自动维护以下四个数据文件：
* `vocab_stats.json`: 存储所有单词的测试数据、错误率分布、记忆间隔（Interval）及下一次艾宾浩斯复习时间戳。
* `study_logs.json`: 记录每次练习的历史时间、题型模式、错题数以及**精确用时**。
* `vocab_config.json`: 持久化锁定上一次的词库勾选组合。
* `custom_lists.json`: 存放通过智能导入器导入的自定义小节词库。
